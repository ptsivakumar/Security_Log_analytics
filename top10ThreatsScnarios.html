Top 10 Threat Scenarios Template

1. Unauthorized Privilege Escalation

High Risk

Description: An attacker gains initial low-level access and attempts to elevate their privileges to an administrator.

Expected Log Evidence

Log Source Expected Events
IDP (Okta/AAD) Logs showing failed attempts to change user roles or groups.
Azure APIM Requests with a low-privilege token accessing admin API endpoints.
Database Attempts to run privileged queries or permission changes on tables.
AKS Cluster API server logs showing attempts to create privileged pods.

2. Web Application Attack (SQL Injection/XSS)

High Risk

Description: An attacker exploits a vulnerability in a web application to steal data or compromise the system.

Expected Log Evidence

Log Source Expected Events
F5 / Azure APIM Logs with suspicious request parameters, malicious query strings, or unusual character sequences.
Container Application logs showing errors from the database or unexpected query syntax.
Database Errors related to malformed queries.

3. Data Exfiltration

High Risk

Description: An attacker with legitimate access (or an insider threat) attempts to steal a large volume of sensitive data.

Expected Log Evidence

Log Source Expected Events
Database An unusual number of SELECT queries or reads on sensitive tables.
Azure APIM API calls to a data export endpoint followed by a large response size.
F5 Outbound traffic logs showing a large volume of data being sent to an external IP.

4. Brute-Force/Credential Stuffing

High Risk

Description: An attacker attempts to guess user passwords or uses a list of stolen credentials to gain unauthorized access.

Expected Log Evidence

Log Source Expected Events
F5 High volume of login requests from a single source IP.
IDP (Okta/AAD) Repeated failed login attempts for a user or across many users.
Azure APIM High number of 401 Unauthorized responses for authentication endpoints.
